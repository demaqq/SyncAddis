Internal Project & Research

Featured Project: Digital ID Integration

Statement of Work (SOW): Integrate the new national Digital ID system to auto-fill applications, reducing data entry errors and verification time.

Key Deliverables:

  • API connection to Digital ID database.
  • "Login with Digital ID" button on citizen portal.
  • Auto-fill functionality for all application forms.

Acceptance Criteria:

  • System successfully retrieves and populates user data in < 1.5 seconds.
  • Data entry errors are reduced by at least 40%.
  • User satisfaction with application process increases by 10 points.

Research & Planning

All initiatives begin with a clear research plan to ensure efficient utilization of resources.

Research Plan (Step 1):

  • Objective: To determine the impact of Digital ID integration on application processing time.
  • Comparative Question: "Is there a significant difference in processing time between citizens using Digital ID and those using manual entry?"
  • Variables: (1) Processing Time (in hours), (2) Entry Method (Digital vs. Manual).
  • Population: All new applicants in Bole and Yeka sub-cities.
Note: All external data sources are vetted using the CRAP Model (Currency Reliability Authority Purpose).
